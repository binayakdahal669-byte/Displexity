Getting Started

Displexity is a modern programming language designed for simplicity, power, and cross-platform deployment.

Installation

// Run the installer or add bin/ to PATH
disp --version
// Output: disp 1.2.0

Hello World

// hello.dis
print "Hello, World!"

Compile and Run

// Default: creates .disp package
disp hello.dis

// Create native executable
disp hello.dis -emit-exe

// Create TUI bytecode (cross-platform)
disp hello.dis -emit-tui

// Run .disp or .tui files
disp run hello.disp
disp run hello.tui

TUI Executables

TUI (Terminal User Interface) executables are cross-platform bytecode files that run on any system with the Displexity runtime.

Creating TUI Files

// Compile to TUI bytecode
disp myapp.dis -emit-tui

// Run TUI file
disp run myapp.tui

TUI Features

  • Cross-platform: runs on Windows, Linux, macOS
  • No dependencies: self-contained bytecode
  • Terminal UI: cursor positioning, colors, input
  • Small size: compressed bytecode format

TUI Functions

// Clear screen
clear()

// Position cursor
gotoxy(10, 5)

// Set text color (0-15)
set_color(14)  // Yellow

// Get key press (non-blocking)
int key = get_key()

// Sleep for milliseconds
sleep(1000)

Standard Library

Math (math.disll)

  • abs(x) - Absolute value
  • sqrt(x) - Square root
  • pow(x, y) - Power
  • sin(x), cos(x), tan(x) - Trigonometry
  • floor(x), ceil(x) - Rounding
  • random() - Random number 0-1

Graphics (graphics.disll)

  • window_create(title, w, h)
  • clear(), gl_swap()
  • draw_rect(x, y, w, h)
  • draw_circle(x, y, r)
  • draw_triangle(x1, y1, x2, y2, x3, y3)
  • set_color(r, g, b)
  • load_image(path), draw_image(img, x, y)

TUI (tui.tuy)

  • clear() - Clear terminal
  • gotoxy(x, y) - Position cursor
  • set_color(c) - Set text color
  • get_key() - Get key press
  • hide_cursor(), show_cursor()

System (system.disll)

  • sleep(ms)
  • getenv(name)
  • exec(cmd)
  • exit(code)
  • file_read(path), file_write(path, data)

Network (network.disll)

  • http_get(url)
  • http_post(url, data)
  • socket_connect(host, port)

Crypto (crypto.disll)

  • md5(data), sha256(data)
  • base64_encode(data), base64_decode(data)
  • aes_encrypt(data, key), aes_decrypt(data, key)

Audio (audio.disll)

  • load_sound(path)
  • play_sound(sound)
  • stop_sound(sound)
  • set_volume(level)

C to Displexity Transpiler

Convert existing C code to Displexity syntax.

Usage

// Transpile C file to Displexity
disp transpile hello.c -o hello.dis

Supported Conversions

  • Function definitions
  • Variable declarations
  • Control flow (if/else, while, for)
  • printf → print
  • scanf → input
  • Operators (&&→and, ||→or)

Example

// C code (hello.c)
#include <stdio.h>
int main() {
    printf("Hello, World!\n");
    return 0;
}

// Transpiled (hello.dis)
func main() {
    print "Hello, World!\n"
    return 0
}

IDE Features

The Displexity IDE (dispe) is a Neovide-based editor with full language support.

Features

  • Syntax highlighting for .dis, .dish, .tuy files
  • Floating autocomplete suggestions
  • Keyword completion for Displexity
  • Multi-language support (C, Python, JS, etc.)
  • Integrated terminal
  • File explorer
  • Fuzzy file finder

Keyboard Shortcuts
