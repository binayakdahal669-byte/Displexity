# Displexity Language Installer Package 
 
This package contains multiple installation options for Displexity: 
 
## Installation Options 
 
1. **GUI Installer** (gui_installer.exe) 
   - Professional graphical installer 
   - Mouse and keyboard support 
   - Component selection 
   - Written in Displexity itself! 
 
2. **Text Installer** (installer_simple.exe) 
   - Command-line installer 
   - Quick and simple 
   - Also written in Displexity! 
 
3. **Wizard Installer** (create_installer_wizard.bat) 
   - Windows batch script 
   - Interactive setup 
   - Automatic PATH configuration 
 
## Quick Start 
 
1. Run `install.bat` to choose installation method 
2. Or run any installer directly 
 
## What Gets Installed 
 
- Displexity compiler (disp.exe) 
- Standard libraries (.disll files) 
- Header files (.dish files) 
- Example programs 
- Documentation 
- System PATH integration 
- File associations 
 
## System Requirements 
 
- Windows 10/11 (64-bit recommended) 
- 100MB free disk space 
- Administrator privileges (for system integration)
