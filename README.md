# DISP Programming Language

A modern, simplified programming language with integrated compiler and IDE.

## Features
- Simple syntax with arrays, functions, classes
- Direct compilation to ASM/executable
- Built-in graphics API
- HTTP request capabilities
- Cross-platform support (Windows, Linux, Mac, Android)
- Integrated IDE with modern features
- Project system (.disproj)
- Dynamic library system (.disll)
- ISO/IMG generation without external tools

## File Extensions
- `.disp` - Source files
- `.dis` - Executable files
- `.disproj` - Project files
- `.disll` - Dynamic libraries
- `.dish` - Header files

## Quick Start
```bash
./dispcompiler --help
./dispcompiler myfile.disp -o myapp.dis
```
