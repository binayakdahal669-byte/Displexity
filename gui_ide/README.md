# DISP IDE (Neovim)

A preconfigured Neovim setup for DISP development.

## Requirements
- Neovim 0.9+ (https://neovim.io/)
- Git (for plugin installation)
- A Nerd Font (for icons)

## Usage
```
dispide.bat [file]
```

## Keybindings

### File Explorer
- `Space + e` - Toggle file explorer
- `Space + o` - Focus file explorer

### Files & Tabs
- `Ctrl + s` - Save
- `Ctrl + n` - New file
- `Ctrl + o` - Open file
- `Ctrl + p` - Find files
- `Ctrl + f` - Search in files
- `Tab` - Next tab
- `Shift + Tab` - Previous tab
- `Space + x` - Close tab

### Terminal
- `Ctrl + `` ` - Toggle terminal
- `Esc` - Exit terminal mode

### Windows
- `Ctrl + h/j/k/l` - Navigate windows

### Build
- `F5` - Build current DISP file

### Quit
- `Space + q` - Quit all
